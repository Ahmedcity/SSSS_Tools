# SSSS_Tools
Collection of tools for Saint Seiya: Soldiers' Soul

### TDB Reader
Reads TDB files. File must be uncompressed.
- Language: Java

### Soul Save Editor
Reads and edits game's saved data. File must be decrypted.
- Language: VB.NET
